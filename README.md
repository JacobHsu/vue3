## VuePress

[VuePress](https://vuepress.vuejs.org/)
[getting-started](https://vuepress.vuejs.org/zh/guide/getting-started.html)

`yarn init`  
`yarn add -D vuepress` 

docs\vue.js> `yarn s`  
success Generated static files in `docs\.vuepress\dist.`  

docs\vue.js> `yarn g`  
docs\vue.js> `yarn d`
